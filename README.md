# 💖 ZAYNFLIX – A Zayn Malik Tribute WebApp

Welcome to **ZAYNFLIX**, a webapp built with pure love, HTML, CSS, and Vanilla JavaScript — dedicated to the artistry, emotion, and evolution of **Zayn Malik**.  
Every section, every animation, every hue of pink — it’s all crafted to capture the *feeling* of listening to Zayn: calm, poetic, and deeply personal.

## 🎬 About the Project

**ZAYNFLIX** is a personal fan-based project that celebrates Zayn Malik’s music, albums, and visual world.  
It’s a responsive, multi-page webapp that brings together his:
- 🎵 **Albums**
- 🎤 **Singles**
- 🎞️ **Music Videos**
- 💬 **About Page** — a heartfelt tribute  
- 💻 **Contact/Footer** — signed by the creator, *OkuhleM*

The design aesthetic is inspired by soft gradients, dark tones, and a touch of pink magic — because if we’re doing Zayn, we’re doing it *beautifully*.

## 💻 Setup and Installation

If you want to run ZAYNFLIX locally:

```bash
# 1. Clone this repo
git clone https://github.com/OkuhleM/ZaynFlix.git

# 2. Navigate to the folder
cd ZaynFlix

# 3. Install dependencies (if using React)
npm install

# 4. Start the development server
npm start
